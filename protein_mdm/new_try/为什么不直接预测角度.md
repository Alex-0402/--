# 为什么蛋白质侧链角度预测采用离散 Bins分类，而不是直接连续回归？

在构建蛋白质构象生成模型时，不直接让网络输出具体的连续角度数值（例如精确到 63.5°），而是将其划分为多个离散的区间（Bins）进行分类预测，主要基于以下三大核心原因：

## 1. 物理结构的多模态特性 (Multi-modality) 与“均值陷阱”
真实世界中，由于原子间的范德华斥力，侧链二面角（如 $\chi_1, \chi_2$）并不是均匀连续分布的。它们通常会聚集在几个特定的“能量波谷”（例如 $+60^\circ$ 和 $-60^\circ$ 附近），而在 $0^\circ$ 附近因为原子严重重叠（Clash），是一个极高的“能量波峰”。
如果遇到局部结构暗示 $+60^\circ$ 和 $-60^\circ$ 都有可能的情况：
- **连续回归模型**：为了“两头不得罪”、使得整体的 MSE 或 MAE 误差最小化，模型会倾向于输出这两者的平均值，也就是 **$0^\circ$**。结果反而预测出了一个物理上绝对非法、能量极高的错误构象。
- **离散 Bins 分类**：模型可以输出一个双峰分布（即在 $+60^\circ$ 的 Bin 和 $-60^\circ$ 的 Bin 各给 50% 概率，而 $0^\circ$ 给 0% 概率）。这完美保留了物理世界“非此即彼”的真实规律。

## 2. 连续回归 (Regression) 本身的数学局限性
只要让模型输出“一个单一的确定数值”（即回归任务），就意味着模型无法表达“这里有两个同等合理的正确答案”这种不确定性。无论是均方误差 (MSE) 还是平均绝对误差 (MAE)，在多目标正确分布下，其数学最优解往往是条件期望（均值）。连续模型只有极高阶的玩法（如混合密度网络 MDN 或 连续扩散 SDE）才能解开多模态分布，但实现与训练难度极大，极易不收敛。

## 3. 与本项目“自适应推理 (Adaptive Inference)”核心创新的强绑定
这对于本毕设（基于离散掩码的自适应扩散模型）而言是最致命的一点。
- 本项目的最大创新在于：**利用 Top-1 和 Top-2 概率的差值（Margin）作为置信度**，像解数独一样“先填极高置信度的锚点，再填低置信度点”。
- 只有将角度切分成离散的 Bins 并进行分类（如使用 Cross-Entropy Loss），模型经过 Softmax 后才能天然输出明确的**概率分布**。有了各个区间的概率，才能计算 Margin，进而驱动自适应推理路径。
- 如果用连续回归直接输出一个 $65.4^\circ$，由于没有其他选项的概率，根本无法衡量模型对这个预测的“犹豫程度”，自适应推理也就无从谈起。

**总结**：将连续角度转化为离散 Bins 并非精度的妥协，而是规避物理“惩罚”的智慧。它不仅完美迎合了旋转异构体的能量分布，更是驱动本课题中“由易到难”自适应推理不可或缺的前提条件。
